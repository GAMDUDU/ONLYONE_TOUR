# 스프링 프로젝트
  스프링 프레임워크를 사용하여 팀원들과 여행사이트를 만드는 것으로 제작
  6명을 인원으로하여 각 파트에 나눠서 작업진행하였습니다.
  작업기간은 약 한달정도 소요가 되었으며,
  기획 및 깃&깃허브를 이용하여 팀원과 프로젝트 공유
  DB설계하는 시간까지 포함되어 있습니다.
  
  그중 제가 구현한 부분에 대해서 GIF형식으로 소개드리며,
  어떤식으로 작업을 진행하였는지 정리하겠습니다.

## 스프링 프로젝트 구현 목록
* 로그인
* 회원가입
* 아이디/비밀번호 찾기
* 상품검색
* 관리자 메인페이지 / 관리자 로그인
* 개인정보 수정 및 탈퇴
